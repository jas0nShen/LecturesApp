# 微信小程序首发提审材料

## 建议版本

- 版本号：`0.1.0`
- 项目阶段：六校授课硕士课程规划首发版
- 发布模式：体验版和正式版使用随包离线数据，不依赖开发环境服务

## 小程序简介

面向香港高校授课硕士学生的课程规划工具。首发支持香港大学、香港中文大学、香港科技大学、香港理工大学、香港城市大学和香港浸会大学的 Programme 索引浏览，并为已开放课程组的 Programme 展示课程结构。用户可保存自己的 Programme，查看资料状态、本机数据说明，并使用课程收藏、Study Plan 和备份功能辅助规划。

## 审核说明

本版本提供六校授课硕士 Programme 浏览与个人课程规划功能。Programme 与课程资料来自公开资料和本地整理流程，应用内“数据状态”页面展示数据覆盖范围、资料来源和发布模式。课程组尚未开放的 Programme 只展示索引与来源，不生成毕业判断。

毕业检查与课程规划仅用于学习规划参考，不代表任何大学的官方审核结果。正式选课、毕业要求、课程开设、学分转换和最终审核结果均以学校官网、选课系统和学校正式通知为准。

所有用户资料、收藏、已修记录、课程笔记和 Study Plan 均保存在微信本地存储。本版本不要求登录，不收集手机号、定位、通讯录、相册、相机或身份证明，不向开发者服务器上传个人资料。

建议审核路径：

1. 首页点击“开始选择”；
2. 选择“授课硕士”，切换学校并选择任一 Programme；
3. 进入 Programme 详情，点击“设为我的 Programme”；
4. 底部进入“课程”，查看已开放课程组或“课程清单待开放”状态；
5. 底部进入“毕业检查”，查看 Programme 资料状态与下一步提示；
6. 底部进入“我的”，查看“首发数据范围”、“数据状态”和“数据与隐私”；
7. 可继续测试课程收藏、Study Plan、复制备份和从剪贴板恢复。

## 后台隐私行为核对

当前代码实际使用：

- 微信本地存储：保存资料、收藏、已修记录、计划、搜索历史和笔记；
- 剪贴板写入：复制课程链接、培养方案链接、Study Plan 和备份文本；
- 剪贴板读取：仅在用户主动点击“从剪贴板恢复”后读取备份；
- 网络请求：仅开发版可能连接本机调试服务；体验版和正式版使用离线数据；
- 分享：用户主动点击课程详情页“分享给好友”后触发。

当前代码不使用：

- 微信登录、用户头像昵称、手机号；
- 定位、地址、通讯录；
- 相机、相册、麦克风；
- 广告、支付或第三方统计 SDK。
