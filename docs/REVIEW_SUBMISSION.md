# 微信小程序首发提审材料

## 建议版本

- 版本号：`0.1.0`
- 项目阶段：香港大学课程规划 MVP
- 发布模式：体验版和正式版使用随包离线数据，不连接 localhost

## 小程序简介

面向香港大学学生的课程规划工具，可查询课程与官方开课资料，记录收藏、已修课程和 Study Plan，并查看毕业要求进度参考。

## 审核说明

本版本提供香港大学课程查询与个人课程规划功能。课程和开课资料来源于香港大学公开网页，本地数据页面提供来源链接及核验日期。毕业检查仅用于学习规划参考，不代表香港大学官方审核结果。

所有用户资料、收藏、已修记录、课程笔记和 Study Plan 均保存在微信本地存储。本版本不要求登录，不收集手机号、定位、通讯录、相册、相机或身份证明，不向开发者服务器上传个人资料。

建议审核路径：

1. 首页进入“课程”；
2. 切换“官方开课”，打开任一课程详情；
3. 测试收藏、标记已修和加入 Study Plan；
4. 底部进入“毕业检查”；
5. 底部进入“我的”，查看“数据状态”和“数据与隐私”。

## 后台隐私行为核对

当前代码实际使用：

- 微信本地存储：保存资料、收藏、已修记录、计划、搜索历史和笔记；
- 剪贴板写入：复制课程链接、培养方案链接、Study Plan 和备份文本；
- 剪贴板读取：仅在用户主动点击“从剪贴板恢复”后读取备份；
- 网络请求：仅开发版尝试连接本机 API；体验版和正式版关闭网络请求并使用离线数据；
- 分享：用户主动点击课程详情页“分享给好友”后触发。

当前代码不使用：

- 微信登录、用户头像昵称、手机号；
- 定位、地址、通讯录；
- 相机、相册、麦克风；
- 广告、支付或第三方统计 SDK。
